Como testar:
1. Descompacte o ZIP.
2. Abra o arquivo index.html no navegador.

Como publicar grátis:
Opção 1: Cloudflare Pages
- Crie conta em cloudflare.com
- Vá em Workers & Pages > Create > Pages
- Faça upload da pasta ou conecte ao GitHub

Opção 2: GitHub Pages
- Crie um repositório no GitHub
- Envie todos os arquivos
- Ative Settings > Pages > Deploy from branch
